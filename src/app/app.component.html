from

from() nos permite crear Observables a partir de arrays, promesas, objetos, strings etc. Suena bastante parecido a of(). from() emite los elementos contenidos dentro del array, mientras que of() emite el array entero. Podríamos decir que from() tiene un efecto de flattening o ‘aplanamiento’: coge un array, string u objeto y lo ‘aplana’, emitiendo los elementos que contiene
